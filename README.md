# Minecraft 模组汉化资源包仓库

最后更新时间：2025-12-18 20:30:12 UTC+8

## 简介

此仓库存储 Minecraft 模组汉化资源包，由 GitHub Actions 每两小时自动更新。

## 自动更新

此仓库使用 GitHub Actions 自动执行以下任务：
1. 每两小时检查并下载最新的汉化资源包
2. 自动更新此 README 文件的时间戳
3. 自动提交并推送更改

## 文件说明

- `crawler.py`: 爬虫脚本，用于从源网站下载资源包
- `resource_pack/`: 存储下载的资源包文件
- `sha256/`: 存储文件的 SHA256 校验和
- `.github/workflows/crawler.yml`: GitHub Actions 工作流配置

## 手动触发
